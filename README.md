# Rufus Web Controller

Web interface for controlling Rufus AI Robot via WiFi.

## Deployment

This repo is deployed on Vercel. Any changes pushed to main will auto-deploy.

## Setup

1. Open the deployed Vercel URL
2. Enter your Pi's IP address (e.g., `http://192.168.1.100:5001`)
3. Click "Save & Connect"

## Requirements

- Raspberry Pi running `pi_api_server.py` on port 5001
- Pi and device on same WiFi network
